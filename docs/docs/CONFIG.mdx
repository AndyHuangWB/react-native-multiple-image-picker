# Configuration

## Core

### MediaType

Type of media to be displayed in the picker.

```typescript
type MediaType = 'video' | 'image' | 'all'
```

- **Type**: string
- **Default**: 'all'
- **Required**: No
- **Options**:

  - `video`: Only show videos
  - `image`: Only show images
  - `all`: Show both videos and images

  ### SelectMode

Mode of selection in the picker.

```typescript
type SelectMode = 'single' | 'multiple'
```

- **Type**: string
- **Default**: 'multiple'
- **Required**: No
- **Options**:
  - `single`: Allow selecting only one item
  - `multiple`: Allow selecting multiple items

### Presentation

Modal presentation style for the picker.

```typescript
type Presentation = 'fullScreenModal' | 'formSheet'
```

- **Type**: string
- **Default**: 'fullScreenModal'
- **Required**: No
- **Platform**: iOS only
- **Options**:
  - `fullScreenModal`: Opens picker in full screen
  - `formSheet`: Opens picker in a form sheet style

## Feature Configurations

### Cropping Options

Configuration for image cropping functionality.

```typescript
type PickerCropConfig = {
  circle?: boolean
}
```

- **Type**: `object`
- **Required**: No
- **Platforms**: iOS, Android
- **Properties**:
  - `circle`: Enable circular crop mask for profile pictures
- **Example**:

```typescript
// Enable basic cropping with default settings
crop: {}

// Enable cropping with circle crop mask
crop: {
  circle: true,
}
```

### Custom Text Labels

Customize button labels and headers for localization and branding.

```typescript
interface Text {
  finish?: string // Done/Complete button
  original?: string // Original image button
  preview?: string // Preview button
  edit?: string // Edit button
}
```

- **Type**: `object`
- **Required**: No
- **Properties**:
  - `finish`: Label for the finish/done button
  - `original`: Label for the original button
  - `preview`: Label for the preview button
  - `edit`: Label for the edit button
- **Example**:

```typescript
text: {
  finish: 'Complete',
  original: 'Original',
  preview: 'Preview',
  edit: 'Edit'
}
```

## UI Configuration

### Colors

#### primaryColor

Primary color for the picker UI elements.

```typescript
primaryColor?: ColorValue
```

- **Type**: ColorValue
- **Default**: '#FB9300'
- **Required**: No
- **Platforms**: iOS, Android
- **Accepts**:
  - Hex strings: '#RGB', '#RGBA', '#RRGGBB', '#RRGGBBAA'
  - RGB/RGBA strings: 'rgb(255, 0, 0)', 'rgba(255, 0, 0, 0.5)'
  - Named colors: 'red', 'blue', etc.
- **Example**:

```typescript
primaryColor: '#FF0000'
primaryColor: 'rgb(255, 0, 0)'
primaryColor: 'red'
```

#### backgroundDark

Background color for dark mode UI elements.

```typescript
backgroundDark?: ColorValue
```

- **Type**: ColorValue
- **Default**: '#2f2f2f'
- **Required**: No
- **Platforms**: iOS, Android
- **Accepts**: Same formats as primaryColor
- **Example**:

```typescript
backgroundDark: '#000000'
backgroundDark: 'rgb(0, 0, 0)'
backgroundDark: 'black'
```

### Quality Settings

#### videoQuality

Video quality setting for the picker.

```typescript
videoQuality?: number
```

- **Type**: number
- **Default**: 1
- **Required**: No
- **Platforms**: iOS, Android
- **Range**: 0 to 1
- **Platform Behavior**:
  - **iOS**: Continuous range (0 = max compression, 1 = no compression)
  - **Android**: Binary (0 = low quality, 1 = high quality)
- **Example**:

```typescript
// iOS: 80% quality
// Android: Low quality (0) or High quality (1)
videoQuality: 0.8 // iOS: 80% quality, Android: Low quality
videoQuality: 1 // Both platforms: Highest quality
```

#### imageQuality

Image quality setting for the picker.

```typescript
imageQuality?: number
```

- **Type**: number
- **Default**: 1
- **Required**: No
- **Platform**: iOS only
- **Range**: 0 to 1
- **Example**:

```typescript
imageQuality: 0.8 // 80% quality
```
